# -customer-churn-prediction-
“customer-churn-prediction”

Project description:

  A data science solution for predicting customer churn for a telecommunication company. 

How to run the code:

  Download the code and import to Google Collab

Key findings summary:

  Fiber optics customer had a higher much higher churn over rate than DSL. Most chrun overs happened  during the first couple of month. Month to month contracts where determined to be a high risk factor suggesting that the increasing the contract lenght beyond month-to-month might reduce churn rate. 
  
Dependencies/ requirements:

  Kagglehub, Pandas, Numpy, matplotlib.pyplot, seaborn, sklearn.model_selection, sklearn.preprocessing, sklearn.compose, sklearn.pipeline,sklearn.linear_model, sklearn.metrics, joblib, 'telco-customer-churn' dataset.
